Circular groove

Stack building blocks: paint the canvas with one foreground (groove) color; composite a lighter and a darker layer made from duplicates of (R1, R2), shifted opposite ways along light—each layer knocks out the inside of the inner ring and the stripe between rings, so only outside the displaced outer ring tints bleed in—and a mirrored pass keeps paint inside the displaced inner disk so the inner wall catches the same cue; then punch the original circles so only the real annulus keeps that stack, filling inner and outer with background, and redraw the rims. Same idea scales to CSS with masks or SVG compositing instead of scratch buffers.

Radii

Lighting

Brighter halos shift along Light from; darker halos the opposite way. Shift magnitudes can differ.

Strength scales halo opacity × layer weight; lightness delta grows with tint.

Halo softness

Blur is applied when compositing displaced halos (ctx.filter). Bloom nudges alpha so spreads don’t look washed out.
